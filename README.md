Resumo do problema a ser resolvido:

o problema a ser solucionado é a identificação e equalização das notas dos alunos, com uma nova ideia para impulsionar os alunos com dificuldades.  

introduzindo formas para minimizar dificuldades de aprendizagem, e superar o desnivelamento inicial.

Fazendo uso de novas métricas de separação e recuperação dos alunos, para que a escola consiga agir diretamente em uma quantidade menor de tempo. 

A instituição necessita de uma maneira que consiga esta  equalização de forma orgânica e sem demandar tanto tempo para adaptação.



Solução proposta peça equipe:

Criação ou utilização de plataformas de fixação como KAHOOT.it, para identificar e auxiliar no tratamento da defasagem de aprendizado dos alunos. Fazendo uso da imersão em um ambiente de competitividade e colaboração.
